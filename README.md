
ly Optimizer

Poly optimizer is a script that optimize stl meshes.

### Version
0.0.1

### Usage

```sh
$ ./poly-opt [input file] [target vertices] > [output file]
```

### Example

```sh
$ ./poly-opt in.stl 10000 > out.stl
```

### Build
```
make
```
